Calcul de la Moyenne et du Mode

1. Moyenne (Moyenne arithmétique)

La moyenne est la somme des valeurs divisée par le nombre total d'observations. Elle est calculée à l'aide de la formule suivante :

Moyenne = (Σ Xi) / N

Où :

  • Xi représente chaque valeur.
  • N est le nombre total d'observations.

Exemple : Pour les valeurs 5, 10, 15, 20, 25, la moyenne est :

Moyenne = (5 + 10 + 15 + 20 + 25) / 5 = 15

2. Mode

Le mode est la valeur qui apparaît le plus fréquemment dans un ensemble de données.

Exemples :

  • Un seul mode : Pour les valeurs 2, 3, 3, 5, 6, 7, le mode est 3.
  • Deux modes : Pour les valeurs 1, 2, 2, 3, 3, 4, 5, les modes sont 2 et 3.
  • Aucun mode : Pour les valeurs 1, 2, 3, 4, 5, il n'y a pas de mode.

Conclusion

La moyenne permet d'obtenir une valeur centrale des données, tandis que le mode identifie la valeur la plus fréquente.

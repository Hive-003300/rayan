Avantages et Retenues sur Salaire

Calcul du Nouveau Salaire Brut Global (SBG) de M. Rida

Avantages en argent : M. Rida bénéficie d'un avantage de 1000 DH en argent.

Calcul du nouveau salaire brut global :

SBG = 8131,21 DH + 1000 DH = 9131,21 DH

Nouveau SBG de M. Rida : 9131,21 DH

Calcul des Retenues sur Salaire : Cotisation CNSS de M. Hamid

Les éléments sociaux de M. Hamid :

  • Salaire de base : 3000 DH
  • Ancienneté : 3 ans
  • Indemnité de déplacement justifiée : 500 DH
  • Indemnité de caisse : 300 DH
  • Prime de fonction : 600 DH
  • Ancienneté : 150 DH

Calcul du Salaire Brut Global (SBG) :

SBG = 3000 DH + 600 DH + 500 DH + 300 DH + 150 DH = 4550 DH

Calcul de la Base Imposable (TB) :

Les éléments exonérés (primes d'aspect social et indemnités justifiées) sont exclus du calcul de la base imposable :

TB = SBG - Éléments exonérés (Indemnités justifiées) TB = 4550 DH - 500 DH = 4050 DH

Calcul de la Cotisation CNSS :

La cotisation à la CNSS est calculée sur la base imposable (TB) avec un taux de 4,48 % :

Cotisation CNSS = 4050 DH × 4,48% = 181,44 DH

Remarque :

Si la base imposable (TB) est supérieure à 6000 DH, elle est plafonnée à 6000 DH. Dans ce cas, la cotisation serait :

Cotisation CNSS = 6000 DH × 4,48% = 268,80 DH

Cotisation CNSS de M. Hamid : 181,44 DH (car le salaire brut global de M. Hamid est inférieur au plafond de 6000 DH).

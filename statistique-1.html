Explication des Calculs Statistiques

1. Effectif Cumulé Croissant (ECC)

L'**ECC** est la somme cumulative des effectifs n_i :

ECC = ∑ n_i

Exemple pour la classe [2600 - 3000[ :

ECC = 175 + 140 = 315

2. Effectif Cumulé Décroissant (ECD)

L'**ECD** est obtenu par :

ECD = N - ECC

Exemple avec N = 645 :

ECD = 645 - 175 = 470

3. Fréquence Relative (%)

La fréquence relative est donnée par :

Fréq = n_i Total ni × 100

Exemple pour [2600 - 3000[ :

175 645 × 100 = 27.13%

4. Fréquence Cumulée Croissante (Fcc)

La fréquence cumulative est donnée par :

Fcc = ∑ Fréq

Exemple :

Fcc = 27.13 + 21.7 + 14.89 = 63.72%

5. Fréquence Cumulée Décroissante (FCD)

La fréquence cumulée décroissante est :

FCD = 100 - Fcc

Exemple :

FCD = 100 - 27,13 = 72.87%
